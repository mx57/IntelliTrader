# План интеграции биржи Bybit в IntelliTrader

## Обзор
Bybit является одной из ведущих бирж для торговли деривативами и спотом. Интеграция позволит расширить возможности бота для пользователей этой площадки. Для реализации будет использоваться библиотека **ExchangeSharp**, которая уже применяется в проекте.

## Шаги реализации

### 1. Анализ текущей архитектуры
- Проверить текущие реализации в `IntelliTrader.Trading`.
- Изучить интерфейс `IExchange` (или аналогичный) в `IntelliTrader.Core`.
- Понять, как реализованы существующие биржи (например, Binance).

### 2. Добавление провайдера Bybit
- Создать новый класс `BybitExchangeProvider` в проекте `IntelliTrader.Trading`.
- Реализовать методы:
    - `GetTickersAsync()` — получение актуальных цен.
    - `GetOrderBookAsync()` — получение стакана для расчета спреда.
    - `PlaceOrderAsync()` — выставление ордеров (Buy/Sell).
    - `GetBalancesAsync()` — получение остатков на счетах.
- Использовать `ExchangeBybitAPI` из ExchangeSharp.

### 3. Регистрация новой биржи
- Обновить `ExchangeFactory` (если имеется) для поддержки типа биржи Bybit.
- Добавить `Bybit` в перечисление `ExchangeType` в `IntelliTrader.Core`.

### 4. Обновление конфигурации
- Добавить поддержку API ключей для Bybit в `appsettings.json` и соответствующие модели конфигурации.
- Реализовать валидацию ключей при запуске.

### 5. Тестирование
- Создать интеграционные тесты для проверки подключения к Bybit API (используя Testnet).
- Проверить корректность расчета спреда и исполнения ордеров в режиме симуляции.

## Ожидаемый результат
Пользователь может выбрать "Bybit" в настройках бота, ввести API ключи и запустить торговые стратегии с использованием данных и ликвидности Bybit.
