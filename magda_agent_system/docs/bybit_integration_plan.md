# План интеграции биржи Bybit в IntelliTrader

## Обзор
Bybit — популярная биржа с поддержкой деривативов и спот-рынка. Интеграция будет опираться на библиотеку **ExchangeSharp**.

## Этапы реализации

### 1. Подготовка
- Проверка совместимости ExchangeSharp с актуальным API Bybit.
- Добавление API-ключей в конфигурацию бота (`PairConfig`).

### 2. Разработка провайдера
- Создание `BybitExchangeProvider`.
- Реализация методов:
    - `GetTickers` (получение котировок).
    - `GetOrderBook` (стакан цен для расчета спреда).
    - `PlaceOrder` (выставление ордеров).
    - `GetBalances` (баланс аккаунта).

### 3. Интеграция в ядро
- Регистрация провайдера в `ExchangeFactory`.
- Обновление перечислений типов бирж.

### 4. Тестирование
- Запуск в режиме симуляции.
- Проверка исполнения ордеров на Bybit Testnet.

## Результат
Бот сможет торговать на Bybit, используя все существующие стратегии (trailing, DCA и др.).
