# План улучшения веб-интерфейса IntelliTrader

## Текущее состояние
Веб-интерфейс построен на ASP.NET Core MVC с использованием Razor-представлений и jQuery/Bootstrap на фронтенде. Дашборд отображает текущие торговые пары, статистику по дням и настройки. Данные обновляются через AJAX-запросы.

## Выявленные недостатки
1. **Отсутствие визуализации в реальном времени**: Все графики (если они есть) статичны или обновляются полной перезагрузкой данных.
2. **Ограниченная аналитика**: Статистика показывает общую прибыль, но не дает наглядного представления о распределении прибыли по монетам или стратегиям в виде диаграмм.
3. **Логирование**: Страница логов просто выводит текст. Нет цветовой кодировки уровней (Error, Warning, Info) или фильтрации на лету.
4. **Управление**: Настройки представляют собой редактор JSON, что мощно, но не всегда удобно для быстрых изменений.

## Предложенные улучшения

### 1. Интерактивные графики (Stats)
- Внедрить библиотеку **Chart.js** или **ApexCharts**.
- Добавить круговую диаграмму (Pie Chart) распределения прибыли по торговым парам.
- Добавить линейный график (Line Chart) изменения баланса за последние 30 дней.

### 2. Улучшенная визуализация логов (Log)
- Добавить цветовую подсветку строк логов в зависимости от уровня важности.
- Реализовать JavaScript-фильтр для поиска по ключевым словам или фильтрации по уровню (Error/Info) без перезагрузки страницы.

### 3. Индикаторы состояния на Дашборде (Dashboard)
- Добавить индикатор прогресса (ProgressBar) для каждого трейлинга.
- Визуализировать расстояние до срабатывания `MaxSpread` (внедренного ранее).

### 4. Мобильная адаптация
- Проверить и улучшить отображение таблиц на узких экранах (использование `DataTable responsive`).

## План реализации (задачи для Jules)
1. **task-log-highlighting**: Внедрить цветовую подсветку уровней логов в `Log.cshtml` и `intellitrader.js`.
2. **task-stats-charts**: Интегрировать Chart.js и добавить графики доходности на страницу статистики.
3. **task-dashboard-spread-indicator**: Добавить визуальный индикатор спреда в таблицу дашборда.
