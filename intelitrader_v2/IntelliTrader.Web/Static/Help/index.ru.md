## Конфигурация

Конфигурация IntelliTrader осуществляется через файлы JSON, расположенные в папке `config`.

### Core

Файл `config/core.json` содержит основные настройки IntelliTrader.

```json
{
  "LogLevel": "Information",
  "RefreshInterval": 60000,
  "DataDirectory": "./Data",
  "LogDirectory": "./Logs"
}
```

- `LogLevel`: Уровень логирования (Trace, Debug, Information, Warning, Error, Critical, None).
- `RefreshInterval`: Интервал обновления данных в миллисекундах.
- `DataDirectory`: Каталог для хранения данных.
- `LogDirectory`: Каталог для хранения логов.

### Web

Файл `config/web.json` содержит настройки веб-интерфейса.

```json
{
  "Port": 7000,
  "EnableHttps": false,
  "CertificatePath": "",
  "CertificatePassword": ""
}
```

- `Port`: Порт, на котором будет работать веб-интерфейс.
- `EnableHttps`: Включить HTTPS.
- `CertificatePath`: Путь к файлу сертификата.
- `CertificatePassword`: Пароль для сертификата.

### Signals

Файл `config/signals.json` содержит настройки для обработки сигналов.

```json
{
  "Enabled": true,
  "Providers": [
    {
      "Name": "ExampleSignalProvider",
      "Type": "Rest",
      "Url": "http://localhost:8080/signals",
      "Interval": 5000
    }
  ]
}
```

- `Enabled`: Включить обработку сигналов.
- `Providers`: Список поставщиков сигналов.
  - `Name`: Имя поставщика.
  - `Type`: Тип поставщика (Rest, Websocket).
  - `Url`: URL поставщика.
  - `Interval`: Интервал запросов к поставщику (для Rest).

### Trading

Файл `config/trading.json` содержит настройки для торговых операций.

```json
{
  "Enabled": true,
  "Exchange": "Binance",
  "ApiKey": "YOUR_API_KEY",
  "ApiSecret": "YOUR_API_SECRET",
  "VirtualTrading": false,
  "DefaultBuyAmount": 100,
  "DefaultSellAmount": 100,
  "MinProfitPercentage": 0.5,
  "MaxLossPercentage": 5.0,
  "DcaLevels": [
    {
      "Percentage": 1.0,
      "Multiplier": 1.5
    }
  ]
}
```

- `Enabled`: Включить торговлю.
- `Exchange`: Используемая биржа (например, Binance, KuCoin).
- `ApiKey`: Ваш API ключ биржи.
- `ApiSecret`: Ваш секретный ключ биржи.
- `VirtualTrading`: Включить виртуальную торговлю (без реальных средств).
- `DefaultBuyAmount`: Сумма покупки по умолчанию.
- `DefaultSellAmount`: Сумма продажи по умолчанию.
- `MinProfitPercentage`: Минимальный процент прибыли для продажи.
- `MaxLossPercentage`: Максимальный процент убытка для стоп-лосса.
- `DcaLevels`: Уровни усреднения стоимости (DCA).
  - `Percentage`: Процент падения цены для активации DCA.
  - `Multiplier`: Множитель суммы покупки для DCA.

## Правила

Правила определяют логику покупки и продажи активов.

### Правила сигналов

Правила сигналов используются для покупки новых пар на основе определенных условий. Это также может быть опционально с трейлингом.

Если включено, то трейлинг сигналов в IntelliTrader является первой стадией покупки, где он будет отслеживать монету на основе заданных вами настроек в течение указанных вами временных интервалов. Если монета соответствует условиям трейлинга, то затем проверяются условия монеты и монета покупается, если они выполняются.

Если трейлинг не включен, покупка осуществляется только на основе заданных условий.

Добавьте `"Action": "Swap"`, если вы хотите, чтобы правило сигнала использовалось только для обмена пар.
Добавьте `"Action": "Arbitrage"`, если вы хотите, чтобы правило сигнала использовалось только для арбитража.

### Торговые правила

Торговые правила применяются к парам, которые у вас уже есть. Трейлинг недоступен для торговых правил. Условия для торговых правил работают так же, как и в правилах сигналов.

Доступные модификаторы для торговых правил - это любые настройки `trading.json`, которые начинаются с `Buy`, `Sell` или `DCALevels`.

Здесь же вы настраиваете свои правила, специфичные для обмена.

## Веб-интерфейс

Для доступа к веб-интерфейсу просто откройте `http://localhost:7000` в вашем браузере (или замените `7000` на настроенный вами порт).

### Строка состояния

Строка состояния содержит информацию о вашем доступном балансе, текущем глобальном рейтинге, трейлинговых покупках/продажах/сигналах и текущем статусе. Наведите курсор на значок статуса (ON), чтобы увидеть последнюю информацию о проверке работоспособности.

### Панель управления

Это экран по умолчанию, с которого запускается IntelliTrader. Таблица показывает все пары, которые у вас в настоящее время есть. Данные автоматически обновляются каждые 5 секунд.

Столбцы можно настраивать. Вы можете изменять их порядок, перемещать и даже удалять те, которые вам не нужны. Есть несколько вариантов экспорта данных: вы можете скопировать их или экспортировать в электронную таблицу.

Вы можете нажать на строки таблицы, чтобы развернуть их и получить доступ к дополнительным опциям. Оттуда вы можете вручную купить, продать или просмотреть текущие настройки пары.

Кнопка журнала отобразит последние 5 строк из файла журнала.

| Пара #2 | Название пары, включая текущий уровень DCA |

| Возраст | Как долго пара удерживается |
| Сумма | Общая сумма пары, которую вы в настоящее время держите |
| Стоимость | Текущая стоимость пары |
| Стоимость покупки | Стоимость покупки пары |
| Цена | Текущая цена пары на бирже |

В нижней части основной области вы найдете полезную информацию о текущих парах: общее количество пар, средняя маржа, средний рейтинг, средний возраст и общая стоимость всех пар вместе.

### Статистика

Здесь вы можете увидеть свою общую прибыль и текущий баланс счета.

Также есть разбивка прибыли по дням, а также другая информация, например, средняя маржа.

Вы можете нажать на количество сделок, чтобы увидеть информацию о завершенных ордерах за этот конкретный день.

Вы также можете открыть страницу анализатора правил, чтобы проверить, как работает каждое правило.

### Настройки

Здесь вы можете временно включить или отключить небольшое подмножество опций бота.

Если вы хотите внести постоянные изменения в настройки, вы можете использовать панель «Дополнительно». Сохранение изменений там вступит в силу немедленно и навсегда. В расширенном редакторе вы можете переключаться между различными режимами редактирования (Дерево, Код, Форма, Текст, Просмотр).

На этой странице также доступны три опции:

- Перезапустить службы: это перезапустит все службы бота (ядро, торговля, сигналы, правила и т. д.). Полезно, когда что-то пошло не так, и у вас нет доступа к вашей машине / VPS.
- Обновить учетную запись: вы можете использовать эту опцию, если вы совершили ручную сделку на бирже и хотели бы немедленно обновить свою учетную запись. Однако помните, что существует периодическое обновление учетной записи (по умолчанию каждые 6 минут), поэтому это редко необходимо.
- Выйти: выйти из IntelliTrader

## Уведомления

### Telegram

В файле `config/notification.json` измените `*Enabled*` на `true`, установите `*TelegramBotToken*` на токен вашего бота и `*TelegramChatId*` на ваш идентификатор чата, чтобы включить уведомления Telegram. Установите `*TelegramAlertsEnabled*` на `*false*`, если вы не хотите получать оповещения с уведомлениями.

Поговорите с `@botfather`, чтобы создать нового бота, а затем поговорите с `@FalconGate_Bot`, чтобы получить свой идентификатор чата Telegram (введите `/get_my_id`).

## Проверки работоспособности

Проверка работоспособности - это периодический тест всех служб бота, чтобы убедиться, что каждая из них работает бесперебойно и без перебоев. Торговля приостанавливается, если хотя бы одна из проверок работоспособности не проходит, пока она снова не пройдет. Если уведомления включены, вы получите уведомление в обоих случаях.

## Устранение неполадок

### IntelliTrader не запускается или вылетает при запуске

Пожалуйста, убедитесь, что ваша система соответствует всем требованиям и что у вас есть все необходимые компоненты. Также проверьте, нет ли ошибок в файле `log/*-general.log`.

### IntelliTrader не торгует или ведет себя странно в целом

Опять же, проверьте файл журнала на наличие ошибок, очень вероятно, что там будет подсказка.
