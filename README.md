<img src="http://intellitrader.io/img/logo.png" alt="logo" width="40" height="40"> IntelliTrader
===========

IntelliTrader is a fully featured, highly configurable trading bot for cryptocurrency exchanges.
IntelliTrader — это полнофункциональный, высоконастраиваемый торговый бот для криптовалютных бирж.

Overview / Обзор
-------------

IntelliTrader is completely free, so please use it at your own risk. Always start by [virtual trading](#virtual-trading) first.
Проект является бесплатным и предоставляется «как есть». Пожалуйста, используйте его на свой страх и риск. Всегда начинайте с виртуальной торговли.

##### Main Features / Основные возможности

* **Cross-platform / Кроссплатформенность**: Windows, Linux, MacOS.
* **Highly configurable / Высокая гибкость**: настройте каждую деталь вашей стратегии.
* **Autonomous Evolution / Автономное развитие**: Integrated with **Magda Agent (Jules)** for self-improvement and automated coding.
* **Portable Releases / Портативные сборки**: Automatic builds for Windows and Linux via GitHub Actions.

Getting Started / Начало работы
-------------

#### Prerequisites / Требования
Download and install .NET Core Runtime 2.1.
Скачайте и установите .NET Core Runtime 2.1.

#### Building and Releases / Сборка и релизы
Download ready-to-use portable builds from the [Releases page](https://github.com/jazzonaut/IntelliTrader/releases).
Скачайте готовые портативные сборки со страницы релизов.

To build manually / Для ручной сборки:
1. `git clone`
2. `git submodule update --init --recursive`
3. Run `Publish.bat` (Windows) or `Publish.sh` (Linux).

#### Autonomous Agent (Jules) / Автономный агент
The bot evolves itself using the Jules loop. See [AGENTS.md](AGENTS.md) for details.
Бот развивает себя сам через цикл Jules. Подробности в [AGENTS.md](AGENTS.md).

- **Binance** (Full support / Полная поддержка)
- *Others in progress / Другие в разработке*
